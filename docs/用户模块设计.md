# 接口设计

## 1. 用户登录 [Login]

---

### 接口说明

用于用户身份验证并获取 JWT 令牌。

### 登录接口

- **Method**: `POST`
- **URL**: `/api/auth/login`

### 请求参数

| 字段       | 类型     | 必填 | 说明  |
|:---------|:-------|:---|:----|
| username | string | 是  | 用户名 |
| password | string | 是  | 密码  |

### 返回结果
```json
{
    "code": 200,
    "data": { "token": "..." },
    "msg": "操作成功"
}
```
