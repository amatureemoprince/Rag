# DeepLink

> 检索增强生成（RAG）是指对大语言模型输出进行优化，使其能够在生成响应之前引用训练数据来源之外的权威知识库。
> 大语言模型（LLM）用海量数据进行训练，使用数十亿个参数为回答问题、翻译语言和完成句子等任务生成原始输出。
> 在 LLM 本就强大的功能基础上，RAG 将其扩展为能访问特定领域或组织的内部知识库，所有这些都无需重新训练模型。
> 这是一种经济高效地改进 LLM 输出的方法，让它在各种情境下都能保持相关性、准确性和实用性。

不论是在工作、学习、创作的过程中，搭建一个专属于自己的知识库都是一件值得的事情，于是有了这个项目。

通过上传私有文档，此项目不仅可以帮助用户更快地搭建自己的知识库，而且可以让ai的回答更加有针对性，更符合你的预期。

# 功能模块

## 用户管理模块

核心功能如下：

- 用户注册
- 用户登录
- 用户密码加密
- 权限控制

在实际场景中，需要根据用户的角色和级别对数据访问权限的控制，这也是一个难点，此项目采用`RBAC`实现对不同角色的权限区分，`标签`实现数据访问
权限隔离。将这两种方式结合可以实现“横向”和“纵向”上的权限隔离。

密码采用`BCrypt`加密，用户共分为`普通用户`和`管理员`。

`普通用户`可以上传文件，访问公共文件；`管理员`可以管理知识库，标签。
